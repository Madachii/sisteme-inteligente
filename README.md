# Predictia Pretului pentru CPU

## 1. Motivatia proiectului
Avand in vedere structura complexa a unui procesor si toate specificatile vizible ale acestora, pretul sau nu este intotdeauna usor de justificat printr-o simpla privire, ca un exemplu clasic, frecventa mai mare nu inseamna intotdeauna performanta mai buna, asa ca motivatia personala in acest proiect este de a masura cat cantareste fiecare specificatie al unui procesor si a calcula un pret final care este indiferent de variabile care nu cotribuie strict la performanta procesorului, cum ar fi generatia.

## 2. Scopul proiectului
Scopul principal al proiectului este dezvoltarea unui program capabil sa estimeze pretul corect de piata al unui procesor, bazandu-se pe performanta dovedita in benchmark-uri si pe caracteristicile fizice.

## 3. Link catre date
[**CPU Benchmarks Compilation**](https://www.kaggle.com/datasets/alanjo/cpu-benchmarks): Motivul alegeri acestui dataset constituie campul __cpuMark__ impreuna cu campul __price__. 

## 3. Tehnologii Planificate
* **Limbaj de programare:** Python
* **Biblioteci principale:** `pandas`, `numpy`, `matplotlib`
